### Additional Information

**Why do you need this certificate?**
```
We need to sign our Windows application to provide users with a trusted installation experience.
Code signing helps prevent security warnings and ensures users that our software hasn't been
tampered with. As an open-source project, we want to maintain transparency and security for
our community.
```

**How will the certificate be used?**
```
The certificate will be used to sign Windows executables (.exe) and installers (.msi)
distributed through GitHub Releases. We use GitHub Actions for automated builds and releases,
and the certificate will be securely stored as GitHub Secrets for CI/CD automation.
```

## 注意事项

1. 确保 GitHub 仓库是公开的
2. 确保有明确的开源协议文件（LICENSE）
3. 确保项目有活跃的提交记录
4. 准备好接收验证邮件
5. 可能需要提供额外的项目证明材料

## 审核时间

- 通常 3-5 个工作日
- 可能需要补充材料
- 保持邮箱畅通

## 证书有效期

- 1 年
- 到期前 30 天可以续期
- 续期流程与首次申请相同
